# tinyAPP

Tiny app is a full stack web application built with Node and Express that allows user to shorten long URLS. 


##Dependencies
    -bcryptjs
    -body-parser
    -cookie-parser
    -cookie-session    
    -ejs
    -express
    
    ## Getting Started 
    - install all dependices using npm install
    - run the server by using "node express_server.js" comand
